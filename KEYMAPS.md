# Keymaps

Quick reference for yabai + skhd keybindings.

## Window Focus

| Keybind | Action |
|---------|--------|
| `alt - j` | Focus window south |
| `alt - k` | Focus window north |
| `alt - h` | Focus window west |
| `alt - l` | Focus window east |

## Display Focus

| Keybind | Action |
|---------|--------|
| `alt - s` | Focus display west |
| `alt - g` | Focus display east |

## Window Swap

| Keybind | Action |
|---------|--------|
| `shift + alt - j` | Swap window south |
| `shift + alt - k` | Swap window north |
| `shift + alt - h` | Swap window west |
| `shift + alt - l` | Swap window east |

## Window Warp (move + split)

| Keybind | Action |
|---------|--------|
| `ctrl + alt - j` | Warp window south |
| `ctrl + alt - k` | Warp window north |
| `ctrl + alt - h` | Warp window west |
| `ctrl + alt - l` | Warp window east |

## Move Window to Display

| Keybind | Action |
|---------|--------|
| `shift + alt - s` | Move window to display west |
| `shift + alt - g` | Move window to display east |

## Resize Mode

| Keybind | Action |
|---------|--------|
| `alt - r` | Enter resize mode |
| `h` | Resize left (-20px) |
| `j` | Resize bottom (+20px) |
| `k` | Resize top (-20px) |
| `l` | Resize right (+20px) |
| `escape` / `return` | Exit resize mode |

| `ctrl + alt - s` | Start yabai |
| `ctrl + alt - r` | Restart yabai |
